# 剣暎ゴシック (KenEiGothic)
源暎ゴシックをTTF化しただけのプロジェクトです. PowerPointへの埋め込みといったことが可能です.

なお, 源暎ゴシックMはもともとTTFで提供されているため, 本リポジトリには含まれません.
